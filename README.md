## 🚀 프로젝트 배경

운영기획팀(운영 관리·이벤트 기획·회계 파트)은 회원 관리, 게시글·댓글 모니터링, 포인트 정산, 시험 후기 검수, 이벤트 기획 등 커뮤니티 운영을 담당합니다.<br/>
이 과정에서 늘어나는 회원 데이터와 콘텐츠를 효율적으로 관리하기 위해 관리자 전용 페이지를 개발했습니다.

<br />

## 🔧 사용 기술

- 코어: `React`, `TypeScript`
- 상태 관리: `TanStack Query`
- 스타일링: `tailwind`, `shadcn/ui`
- 빌드 도구: `Vite`
- 패키지 매니저: `npm`
- 배포: `Cloudflare Pages`
